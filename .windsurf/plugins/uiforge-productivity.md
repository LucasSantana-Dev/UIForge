# Siza Productivity Plugin

A comprehensive plugin for Siza development productivity.

## Plugin Contents:

### Commands
- `/deploy` - Smart deployment with quality gates
- `/component` - React component generator
- `/security` - Security audit automation
- `/performance` - Performance analysis
- `/test` - Comprehensive test suite
- `/git` - Smart Git workflow
- `/context` - Context optimization

### Subagents
- Frontend Developer Specialist
- Backend API Developer
- Database Schema Specialist
- E2E Testing Specialist
- Security & Performance Specialist
- MCP Tools Developer
- Code Generation Templates
- DevOps & Deployment Specialist

### Hooks
- Pre-command security validation
- Post-tool auto-formatting
- Session-end cleanup and reporting

### MCP Servers
- brave-search - Web search capabilities
- exa - Enhanced search
- memory - Context memory
- sequential-thinking - Multi-step reasoning
- tavily - Research automation
- snyk - Security scanning

```bash

# Work on feature...
```
