# Test Everything

Run comprehensive test suite for Siza application.

## Test Categories:
1. Unit Tests - Jest/Testing Library
2. Integration Tests - API endpoints
3. E2E Tests - Playwright browser automation
4. Component Tests - React component testing
5. Security Tests - Snyk vulnerability scanning
6. Performance Tests - Lighthouse audits

## Test Commands:
/test unit - Run unit tests only
/test e2e - Run end-to-end tests
/test integration - Run API integration tests
/test security - Run security vulnerability scan
/test performance - Run performance audit
/test all - Run complete test suite

## Coverage Requirements:
- Unit test coverage: > 80%
- E2E critical paths: 100%
- Security scan: No critical vulnerabilities
- Performance score: > 90

## Test Environment:
- Node.js version: Latest LTS
- Browser: Chromium (Playwright)
- Database: Test instance (isolated)
- API: Local development server

## Reporting:
- Test results summary
- Coverage report
- Security scan results
- Performance metrics
- Failed test details

## Usage:
/test - Run all tests with coverage
/test watch - Run tests in watch mode
/test debug - Run tests with debugging
/test ci - Run tests for CI/CD pipeline
