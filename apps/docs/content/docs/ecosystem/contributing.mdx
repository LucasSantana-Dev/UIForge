## Development Setup

```bash
git clone https://github.com/YOUR_USERNAME/siza.git
cd siza
npm install
npm run dev
```

## Commit Conventions

[Conventional Commits](https://www.conventionalcommits.org/) required. Sentence-case subjects:

```
feat: Add template pack generation tool
fix: Resolve webhook plan resolution for team tier
refactor: Extract quality scoring into separate module
chore: Update CI workflows to Node 22
docs: Add MCP integration guide
```

## Code Standards

- TypeScript strict mode
- Functions under 50 lines, cyclomatic complexity under 10
- Line width under 100 characters
- Self-documenting code — no comments unless necessary
- Security-first: validate inputs, sanitize outputs

## Testing

- Coverage target: **80%+**
- Test business logic, not trivial getters/setters
- Edge cases and error conditions

```bash
npm run test        # Unit tests
npm run test:e2e    # E2E tests (siza only)
npm run lint        # Linting
npm run type-check  # TypeScript checking
```
