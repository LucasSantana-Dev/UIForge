Siza's MCP server exposes 17 AI generation tools to any IDE that supports the [Model Context Protocol](https://modelcontextprotocol.io/).

## Install

### NPX (recommended)

```bash
npx @forgespace/ui-mcp
```

### Docker

```bash
docker run -p 3100:3100 ghcr.io/forge-space/ui-mcp:latest
```

### From source

```bash
git clone https://github.com/Forge-Space/ui-mcp.git
cd ui-mcp
npm install && npm run build
node dist/index.js
```

## IDE Configuration

### Claude Desktop

Add to `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) or `%APPDATA%\Claude\claude_desktop_config.json` (Windows):

```json
{
  "mcpServers": {
    "siza": {
      "command": "npx",
      "args": ["@forgespace/ui-mcp"]
    }
  }
}
```

### Claude Code

Add to `.mcp.json` in your project root:

```json
{
  "mcpServers": {
    "siza": {
      "command": "npx",
      "args": ["@forgespace/ui-mcp"]
    }
  }
}
```

## Usage

Once connected, ask your AI assistant to use Siza tools:

**Generate a component:**

```
Use siza to generate a React navbar with a logo, navigation links,
and a mobile hamburger menu using Tailwind CSS
```

**Generate a full page:**

```
Use siza to generate a SaaS landing page with hero section,
features grid, pricing table, and footer
```

**Scaffold an application:**

```
Use siza to scaffold a Next.js ecommerce app with product listing,
cart, checkout, and dashboard pages
```

**Refine a component:**

```
Use siza to refine this component: add dark mode support,
improve accessibility, and make it responsive
```

## Tool Parameters

Most tools accept these common parameters:

| Parameter           | Type                                                | Description                                                        |
| ------------------- | --------------------------------------------------- | ------------------------------------------------------------------ |
| `framework`         | `react` \| `vue` \| `svelte` \| `angular` \| `html` | Target framework                                                   |
| `mood`              | string                                              | Design mood (e.g., `bold`, `minimal`, `premium`)                   |
| `industry`          | string                                              | Industry context (e.g., `saas`, `ecommerce`, `fintech`)            |
| `visual_style`      | string                                              | Visual style (e.g., `glassmorphism`, `bento-grid`, `dark-premium`) |
| `component_library` | string                                              | UI library (e.g., `shadcn`, `radix`, `headless-ui`)                |

See [MCP Tools Reference](/docs/api-reference/mcp-tools) for the complete tool specification.
