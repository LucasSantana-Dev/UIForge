# Components

Siza uses [shadcn/ui](https://ui.shadcn.com) built on Radix UI primitives with Tailwind CSS.

## General

| Component | Description |
|-----------|------------|
| **Button** | Primary actions, variants: default, outline, ghost, destructive |
| **Card** | Container for grouped content |
| **Badge** | Status indicators and tags |
| **Avatar** | User profile images with fallback |

## Navigation

| Component | Description |
|-----------|------------|
| **Tabs** | Tabbed content switching |
| **DropdownMenu** | Contextual action menus |
| **Sheet** | Slide-over panel |

## Data Entry

| Component | Description |
|-----------|------------|
| **Input** | Text input with validation |
| **Textarea** | Multi-line text input |
| **Select** | Dropdown selection |

## Feedback

| Component | Description |
|-----------|------------|
| **Toast (Sonner)** | Non-blocking notifications |
| **Progress** | Determinate progress indicator |
| **Dialog** | Modal confirmation and forms |

## Siza-Specific

| Component | Description |
|-----------|------------|
| **SizaBackground** | Ambient dot grid + brand glow |
| **PricingCard** | Billing plan display with Stripe |
| **UsageChart** | Generation usage visualization |
| **MonacoEditor** | Code editor with syntax highlighting |
