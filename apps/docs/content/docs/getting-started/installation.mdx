## Prerequisites

- **Node.js** 20+ (LTS recommended)
- **npm**, **pnpm**, or **bun** as your package manager
- A [Supabase](https://supabase.com) account (for auth and database)
- Optional: [Cloudflare](https://cloudflare.com) account for deployment

## Install from npm

### npm

```bash
npx create-siza-app my-project
cd my-project
npm install
```

### pnpm

```bash
pnpm dlx create-siza-app my-project
cd my-project
pnpm install
```

### bun

```bash
bunx create-siza-app my-project
cd my-project
bun install
```

## MCP Server (standalone)

If you only need the MCP server for AI-powered generation in your IDE:

```bash
npx @forgespace/siza-mcp
```

This runs the MCP server locally. Configure your IDE to connect via stdio.

## Desktop App

Download the latest release from [GitHub Releases](https://github.com/Forge-Space/siza/releases):

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `Siza-{version}-arm64.dmg` |
| macOS (Intel) | `Siza-{version}-x64.dmg` |
| Windows | `Siza-{version}-setup.exe` |
| Linux (AppImage) | `Siza-{version}.AppImage` |
| Linux (deb) | `Siza-{version}.deb` |
